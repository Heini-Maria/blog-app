# Nerdy Facts  = Micro blogging platform

Personal full stack project to practise rest APIs and to understand the relationship between back- and front-end. 
The react application is created from scratch, using webpack.

Nerdy Facts is a micro blogging platform that allows logged in user to creat new posts, delete and edit them. 
User can also like and comment posts created by other users.

Visit the deployed application here: https://sunny-cascaron-e5deb2.netlify.app/

## Technologies used

  * React
  * Webpack
  * Express
  * mySQL
    
## Features

 * Login and user authentication
 * dark & light mode
 * Logged in user can add, like and comment posts
 * User can edit and delete post that she/he created

### Limitations

  * For now, the whole page refresh after editing or adding a post
